# Student Performance Analysis: Attendance vs Marks 🎓

This project explores the relationship between student attendance and exam marks using Python's data science libraries. It serves as a practical demonstration of data manipulation and visualization.

---

## 🚀 Project Overview
The primary goal of this analysis is to visualize the correlation between how often a student attends classes and the final marks they achieve. By using a scatter plot, we can easily identify patterns and trends in the student data.

## 🛠️ Tech Stack & Libraries
The following tools were used to build this project:

* **Python**: The core programming language.
* **Pandas**: Used for structured data handling and creating DataFrames.
* **Matplotlib**: The foundation for creating static, animated, and interactive visualizations.
* **Seaborn**: A high-level interface built on top of Matplotlib for drawing attractive and informative statistical graphics.

## 📊 Visualizing the Data
The script generates a **Scatter Plot** where:
* **X-axis**: Represents the Attendance percentage.
* **Y-axis**: Represents the Marks obtained.

This visualization helps in understanding if a linear relationship exists between these two variables.
